# 📘 Guía de Proyecto — PS  
Desarrollo de Software en Entornos Profesionales (IS1) — Ciclo 6

---

## 🎯 1. Propósito

La presente guía orienta el desarrollo del Proyecto Sello (PS) del curso de Desarrollo de Software en Entornos Profesionales, en el marco del enfoque por competencias del programa.

👉 Este proyecto se desarrolla bajo la guía general:

- [Guía de Proyectos — PS / PI / EPE](../proyectos.md)

El proyecto IS1 consolida el desarrollo de software en un **entorno de trabajo colaborativo**, integrando prácticas profesionales.

---

## 🧠 2. Enfoque

> Se evalúa la capacidad de desarrollar software en equipo, integrando componentes, gestionando cambios y manteniendo coherencia técnica.

El estudiante trabaja sobre un sistema existente o en evolución:

- Comprensión del sistema  
- Organización del trabajo  
- Desarrollo por iteraciones  
- Integración continua  
- Mejora progresiva  

Los entregables representan **evidencias de competencias**, no fases.

---

## 🔍 3. Naturaleza del proyecto

El proyecto:

- Se desarrolla sobre un sistema existente o previamente construido  
- Integra múltiples componentes  
- Simula un entorno profesional de desarrollo  
- Se organiza por roles y tareas  

> ✔ Se prioriza integración, no desarrollo aislado

## 🧩 10. Tipo de sistema

Sistema integrado:

- Puede ser web, móvil o distribuido  
- Basado en proyecto previo  
- En evolución continua  

---

## 🎯 11. Nivel esperado

PS — Nivel intermedio/profesional:

- Trabajo colaborativo real  
- Integración continua  
- Gestión de cambios  
- Sistema funcional en evolución  

---

## ⚠️ 12. Consideraciones

- Trabajo en equipo obligatorio  
- Uso de repositorio (Git) obligatorio  
- Uso de ramas y control de versiones  
- Coordinación entre integrantes  

🚫 No se aceptan:

- trabajo individual aislado  
- código sin integración  
- entregas sin historial  
- falta de coordinación  

---

## 🏁 Mensaje final

> El estudiante desarrolla software en condiciones cercanas a la industria, integrando trabajo en equipo, control de versiones y evolución continua del sistema.
