# 📘 Guía de Proyecto — PS  
Programación Básica (FP) — Ciclo 1

---

## 🎯 1. Propósito

La presente guía orienta el desarrollo del Proyecto Sello (PS) del curso de Programación Básica, en el marco del enfoque por competencias del programa.

👉 Este proyecto se desarrolla bajo la guía general:

- [Guía de Proyectos — PS / PI / EPE](../proyectos.md)

La presente guía define el **alcance técnico del curso**.

---

## 🧠 2. Enfoque

> No se evalúan ejercicios aislados, se evalúa la capacidad de construir una solución funcional.

El estudiante desarrolla una solución progresivamente:

- Definición del problema  
- Diseño lógico  
- Implementación  
- Integración básica  

Los entregables representan **evidencias de competencias**, no tareas aisladas.

---

## 🔍 3. Naturaleza del proyecto

El proyecto:

- Resuelve un problema básico de negocio  
- Se implementa como programa en consola  
- Integra lógica, control y datos  
- Se desarrolla en equipo  

> ✔ No se aceptan programas inconexos o sin sentido aplicado

La evaluación se realiza por competencias:

## 🎤 9. Sustentación

Incluye:

- Explicación del problema  
- Demostración del programa  
- Lógica implementada  
- Respuesta a preguntas  

👉 Evaluada con rúbrica CE0217

---

## 🧩 10. Tipo de sistema

Sistema en consola:

- Programación estructurada  
- Entrada/salida  
- Control de flujo  
- Funciones  

---

## 🎯 11. Nivel esperado

PS — Nivel inicial:

- Programa funcional  
- Lógica correcta  
- Integración básica  
- Explicación clara  

---

## ⚠️ 12. Consideraciones

- Trabajo en equipo  
- Uso de repositorio (si aplica)  
- Cada entrega es requisito para la siguiente  
- Debe existir coherencia  

🚫 No se aceptan:

- ejercicios aislados  
- programas incompletos  
- código sin lógica  

---

## 🏁 Mensaje final

> El estudiante inicia construyendo programas básicos, sentando las bases para el desarrollo de sistemas más complejos en niveles posteriores.
