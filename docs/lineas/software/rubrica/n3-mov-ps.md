# RÚBRICA NIVEL 3 — MOV (Aplicaciones Móviles, Ciclo 6) — PS

## CONTEXTO DE EVALUACIÓN

Se evalúa sobre:
- Producto final del sistema integrado: aplicación móvil funcional, integrada y alineada al sistema completo
- Evidencia: CE023f (aplicación móvil)

## PRINCIPIO

- Evalúa la capacidad de construir e integrar una aplicación móvil profesional como parte de un sistema completo.

## CRITERIOS DE EVALUACIÓN

| Criterio                        | Excelente (18–20)                                                                 | Bueno (15–17)                                   | Regular (13–14)                        | Deficiente (<13)                |
|---------------------------------|-----------------------------------------------------------------------------------|-------------------------------------------------|-----------------------------------------|-------------------------------|
| Integración con el sistema      | La app móvil está completamente integrada y sincronizada con el sistema principal  | Integración funcional con detalles menores       | Integración parcial o con problemas     | No hay integración funcional     |
| Funcionalidad                   | La app cumple todos los requisitos y opera correctamente en todos los escenarios   | Funcionalidad mayormente operativa con detalles menores | Funcionalidad incompleta o con errores | La app no cumple el objetivo     |
| Interfaz de usuario (UI)        | Interfaz intuitiva, profesional y alineada a los requisitos                        | Interfaz funcional con detalles menores          | Interfaz limitada o poco usable         | No hay interfaz o es inusable    |
| Seguridad y desempeño           | Implementa buenas prácticas de seguridad y responde eficientemente                 | Seguridad y desempeño aceptables                | Seguridad o desempeño limitados         | No hay seguridad ni desempeño    |
| Documentación y estilo          | Código y arquitectura bien documentados                                            | Documentación y estilo adecuados                 | Documentación limitada                  | Sin documentación ni estilo      |

## ESTRUCTURA DE LA RÚBRICA

- Integración
- Funcionalidad
- Interfaz
- Seguridad y desempeño
- Documentación

## VALIDACIÓN

- ✔ Alineada a MOV (c6) y EPE (N3)
- ✔ Evalúa integración profesional de la app móvil
- ✔ Es defendible y observable

## FRASE CLAVE

> “El desarrollo profesional de apps móviles se valida cuando la aplicación es parte integral de un sistema completo y seguro.”
