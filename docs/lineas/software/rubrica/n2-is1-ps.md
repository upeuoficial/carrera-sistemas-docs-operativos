# RÚBRICA NIVEL 2 — IS1 (Gestión Técnica y Calidad, Ciclo 6) — PS

## CONTEXTO DE EVALUACIÓN

Se evalúa sobre:
- Producto final del curso: sistema funcional con gestión técnica y control de calidad inicial
- Evidencia: CE0243 (gestión técnica)

## PRINCIPIO

- Evalúa la capacidad de aplicar prácticas de gestión técnica y calidad en el desarrollo de una pieza funcional de software.

## CRITERIOS DE EVALUACIÓN

| Criterio                        | Excelente (17–20)                                                                 | Bueno (13–16)                                   | Regular (10–12)                        | Deficiente (0–9)                |
|---------------------------------|-----------------------------------------------------------------------------------|-------------------------------------------------|-----------------------------------------|-------------------------------|
| Gestión técnica del desarrollo  | Evidencia clara de decisiones técnicas y control de deuda técnica                  | Gestión técnica presente pero parcial           | Gestión técnica limitada                | No hay gestión técnica           |
| Pruebas iniciales               | Pruebas funcionales cubren los módulos principales                                | Pruebas presentes pero con cobertura parcial    | Pruebas limitadas o poco efectivas      | No hay pruebas                  |
| Control de calidad del sistema  | Evidencia de revisión de código y mejora continua                                 | Revisión y mejora parcial                      | Revisión superficial                    | No hay revisión ni mejora        |
| Documentación de calidad        | Documentación clara de criterios y resultados de calidad                          | Documentación parcial o incompleta             | Documentación limitada                  | No hay documentación             |
| Integración de prácticas de calidad | Prácticas de calidad integradas en el ciclo de desarrollo                        | Integración funcional con detalles menores      | Integración parcial o con problemas      | No hay integración de calidad    |

## ESTRUCTURA DE LA RÚBRICA

- Gestión técnica
- Pruebas
- Control de calidad
- Documentación
- Integración de calidad

## VALIDACIÓN

- ✔ Alineada a IS1 (c6)
- ✔ Evalúa calidad y gestión técnica a nivel de pieza funcional
- ✔ Es defendible y observable

## FRASE CLAVE

> “La calidad profesional inicia con la gestión técnica y la mejora continua desde el primer sistema funcional.”
